How it works

Three steps. Zero config. Works with your existing workflow.

1

Install & init

npm install -g @ivenlau/minspect && minspect init. Hooks are installed automatically for Claude Code, Codex CLI, and OpenCode.

$ minspect init
2

Code as usual

Use your AI coding agents normally. minspect captures every edit in the background — no slowdown, no interruptions.

$ claude
3

Inspect & review

Open the dashboard at localhost. See what changed, replay sessions, and revert if needed.

$ open 127.0.0.1:21477
